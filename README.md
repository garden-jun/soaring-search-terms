# soaring-search-terms
데이터크롤링을 이용한 실시간 급상승 아이템 선정 프로그램

사라진 네이버의 실시간 검색어를 직접 구현해보고자 한 프로젝트

아래 이미지는 '음식'에 관련하여 진행(테스트)한 결과이미지입니다.
